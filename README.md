## Installation

```bash
npm i
```

## Creating Environment Variables

Create `.env` file using as reference the `.env.example` file.
```bash
AWS_REGION="sa-east-1"
ACCESS_KEY=""
SECRET_ACCESS_KEY=""
QUEUE_URL=""
DB_URL=""
```

## Building the app

```bash
npm run build
```

## Running the app

```bash
npm start
```
